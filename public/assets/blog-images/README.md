# Blog Images Directory

Place your blog cover images here.

## Recommended image specifications:
- Format: WebP or JPEG
- Size: 1200x630px (for optimal social media sharing)
- Compression: Optimized for web
- Naming: Use descriptive names like `nextjs-14-guide.jpg`

## Usage:
Reference images in your blog data as `/assets/blog-images/filename.jpg`

Example:
```typescript
coverImage: "/assets/blog-images/nextjs-14-guide.jpg"
```
